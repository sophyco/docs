---
title: 'List Transcripts'
openapi: 'GET /api/transcripts/{projectId}/'
---

Retrieve conversation transcripts for a project with optional filtering.

## Path Parameters

- `projectId` (string, required): Project UUID

## Query Parameters

- `conversationID` (string): Filter by conversation ID
- `sessionID` (string): Filter by session ID
- `platform` (string): Filter by platform (whatsapp, webchat, etc.)
- `unread` (boolean): Filter by unread status

## Response

Returns an array of transcript objects.

## Examples

```bash
# Get all transcripts
curl -X GET http://localhost:8000/api/transcripts/{projectId}/ \
  -H "Authorization: Bearer suk_..."

# Filter by platform
curl -X GET "http://localhost:8000/api/transcripts/{projectId}/?platform=whatsapp" \
  -H "Authorization: Bearer suk_..."

# Filter by unread
curl -X GET "http://localhost:8000/api/transcripts/{projectId}/?unread=true" \
  -H "Authorization: Bearer suk_..."
```
