---
title: 'Get Project'
openapi: 'GET /api/organizations/{organization_id}/projects/{id}/'
---

Retrieve a specific project by ID.

<Note>
This endpoint requires authentication with an API token (Bearer token). You can only retrieve projects in organizations you belong to.
</Note>

## Path Parameters

- `organization_id` (uuid, required): Organization ID
- `id` (uuid, required): Project ID

## Response

Returns the project details.

```json
{
  "id": "uuid",
  "organization_id": "uuid",
  "organization_name": "Organization Name",
  "name": "Project Name",
  "created_at": "2025-08-25T14:44:42.414344Z",
  "updated_at": "2025-08-25T14:44:42.414358Z"
}
```

## Fields

- `id` (uuid): Project ID
- `organization_id` (uuid): Organization ID
- `organization_name` (string): Organization name
- `name` (string): Project name
- `created_at` (datetime): Creation timestamp
- `updated_at` (datetime): Last update timestamp
