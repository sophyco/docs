---
title: 'List API Tokens'
openapi: 'GET /api/auth/tokens/'
---

List all API tokens for the authenticated user.

<Info>
Requires authentication with a valid API token (suk_* or ssk_*)
</Info>

## Response

Returns an array of API token objects.

```json
[
  {
    "id": "uuid",
    "token": "suk_...",
    "type": "user|service",
    "name": "string",
    "user_id": "uuid",
    "is_active": true,
    "expires_at": "2026-12-31T23:59:59Z",
    "last_used_at": "2026-04-20T10:30:00Z",
    "is_expired": false,
    "is_valid": true,
    "created_at": "2026-04-20T09:00:00Z",
    "updated_at": "2026-04-20T09:00:00Z"
  }
]
```

## Token Fields

- `token`: The actual token value (suk_* for user, ssk_* for service)
- `type`: Either "user" or "service"
- `is_active`: Whether the token is active
- `expires_at`: Expiration date (null for unlimited)
- `is_valid`: Computed field (is_active && !is_expired)
