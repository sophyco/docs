---
title: 'Get Organization'
openapi: 'GET /api/organizations/{id}/'
---

Retrieve a specific organization by ID.

<Note>
This endpoint requires authentication with an API token (Bearer token).
</Note>

## Path Parameters

- `id` (uuid, required): Organization ID

## Response

Returns the organization details.

```json
{
  "id": "uuid",
  "name": "Organization Name",
  "created_at": "2025-08-25T14:44:21.423505Z",
  "project_count": 2
}
```

## Fields

- `id` (uuid): Organization ID
- `name` (string): Organization name
- `created_at` (datetime): Creation timestamp
- `project_count` (integer): Number of projects in the organization
